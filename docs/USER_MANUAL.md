## 简介

微信AI助手是一款功能完整的桌面应用程序，专为需要高效管理微信消息的用户设计。本应用可以：

- 📥 **读取微信历史聊天记录** - 从PC端微信读取并展示历史消息
- 👁️ **实时监控群聊** - 监听指定群聊的新消息
- 🤖 **AI智能分析** - 利用AI自动生成待办事项
- 📚 **知识库管理** - 将重要消息结构化存储，支持智能检索
- 💬 **AI对话助手** - 与AI进行自由对话，获取智能帮助

## 系统要求

### 最低配置
- **操作系统**: Windows 10 / macOS 10.13 / Linux (Ubuntu 18.04+)
- **内存**: 4GB RAM
- **存储**: 200MB 可用空间
- **微信PC版**: 需要安装并登录微信PC版

### 推荐配置
- **操作系统**: Windows 11 / macOS 12+
- **内存**: 8GB RAM
- **存储**: 500MB 可用空间
- **AI服务**: Ollama本地部署或OpenAI API

## 安装指南

### Windows用户

1. 下载 `微信AI助手-Setup.exe` 安装包
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 从开始菜单或桌面快捷方式启动应用

### macOS用户

1. 下载 `微信AI助手.dmg` 安装包
2. 双击打开DMG文件
3. 将应用拖拽到"应用程序"文件夹
4. 从启动台启动应用

### Linux用户

1. 下载 `微信AI助手.AppImage` 文件
2. 添加执行权限：`chmod +x 微信AI助手.AppImage`
3. 双击或通过命令行运行

## 快速入门

### 第一步：配置微信数据目录

1. 启动应用后，点击左侧导航栏的 **设置** ⚙️
2. 在"微信数据配置"部分，点击 **浏览** 按钮
3. 选择微信数据目录：
   - Windows: `C:\Users\您的用户名\Documents\WeChat Files\您的微信ID\`
   - macOS: `~/Library/Containers/com.tencent.xinWeChat/Data/Library/Application Support/`
4. 点击确认保存

### 第二步：配置AI服务

**选项A：使用本地Ollama（推荐，隐私更好）**

1. 安装 [Ollama](https://ollama.ai/)
2. 下载AI模型：`ollama pull qwen2` 或 `ollama pull llama3`
3. 确保Ollama服务运行中
4. 应用会自动检测本地模型

**选项B：使用OpenAI API**

1. 获取 [OpenAI API Key](https://platform.openai.com/)
2. 在设置页面输入API Key
3. 保存设置

### 第三步：开始使用

配置完成后，您可以：
- 在 **消息管理** 查看历史聊天
- 在 **群聊监控** 实时监听群消息
- 使用 **AI对话** 与助手交流
- 管理 **待办事项**

## 功能详解

### 📊 仪表盘

主页显示关键统计信息：
- 联系人数量
- 群聊数量
- 待办事项数量
- 知识库条目数量

提供快速操作入口，一键进入常用功能。

### 💌 消息管理

**功能特点：**
- 按联系人/群聊分类浏览消息
- 支持搜索功能
- 查看完整聊天历史
- 一键让AI分析聊天内容并生成待办
- 将重要消息添加到知识库

**操作步骤：**
1. 选择"联系人"或"群聊"标签
2. 点击列表中的会话
3. 右侧显示该会话的消息历史
4. 使用底部按钮进行AI分析或知识库操作

### 👁️ 群聊监控

**功能特点：**
- 选择需要监控的群聊
- 实时接收新消息通知
- 消息以时间线形式展示

**操作步骤：**
1. 勾选要监控的群聊
2. 点击"开始监控"
3. 新消息会实时显示在右侧面板
4. 点击"停止监控"结束

### ✅ 待办事项

**功能特点：**
- 手动添加待办事项
- AI从聊天中自动提取待办
- 优先级管理（紧急/普通/低优先）
- 完成状态跟踪
- 筛选查看（全部/待完成/已完成/AI生成）

**操作步骤：**
1. 在输入框输入待办内容
2. 选择优先级
3. 点击"添加"按钮
4. 点击复选框标记完成

### 📚 知识库

**功能特点：**
- 存储重要聊天信息
- 关键词搜索
- AI智能问答

**操作步骤：**
1. 在消息管理页面将消息添加到知识库
2. 使用搜索框查找信息
3. 在"AI智能问答"输入问题，获取基于知识库的回答

### 🤖 AI对话

**功能特点：**
- 自由对话
- 上下文记忆
- 多种AI能力支持

**可以问AI：**
- "帮我总结今天的工作安排"
- "根据这段聊天记录，有哪些需要跟进的事项？"
- "如何礼貌地拒绝一个请求？"

### ⚙️ 设置

**微信数据配置：**
- 选择微信数据目录

**AI服务配置：**
- Ollama服务地址（默认 http://localhost:11434）
- OpenAI API Key（备用）
- AI模型选择

## 常见问题

### Q: 为什么群聊和聊天信息是示例数据？

**A:** 当您首次使用应用或尚未配置微信数据目录时，应用会显示示例数据（演示模式）。这些示例数据用于展示应用的各项功能。要查看真实的微信数据：

1. 进入**设置**页面
2. 在"微信数据配置"部分配置您的微信数据目录
3. 保存后，应用将读取您的真实聊天记录

您可以通过界面上的橙色提示框识别当前是否处于演示模式。

### Q: 为什么看不到微信消息？

**A:** 请检查以下几点：
1. 确认已正确配置微信数据目录
2. 确保微信PC版已登录
3. 微信数据目录中应包含 `Msg` 文件夹

### Q: AI功能不可用怎么办？

**A:** 请确认：
1. 如果使用Ollama：
   - Ollama已安装并运行
   - 至少下载了一个模型（运行 `ollama list` 查看）
2. 如果使用OpenAI：
   - API Key正确且有效
   - 账户有足够余额

### Q: 如何保护隐私？

**A:** 本应用注重隐私保护：
- 所有数据处理在本地完成
- 使用Ollama时，数据完全不上传
- 使用OpenAI时，仅发送必要的分析内容
- 不收集任何用户数据

### Q: 应用占用资源高怎么办？

**A:** 建议：
1. 减少监控的群聊数量
2. 定期清理知识库中不需要的内容
3. 关闭不使用的功能页面

### Q: 如何备份数据？

**A:** 应用数据存储在用户目录下，可通过复制数据文件夹进行备份。
