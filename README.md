## ✨ 功能特性

### 📥 消息管理
- 从PC端微信读取历史聊天记录
- 支持联系人和群聊消息浏览
- 消息搜索功能

### 👁️ 实时监控
- 监听指定微信群聊的实时消息
- 消息时间线展示
- 即时通知

### 🤖 AI智能助手
- **自动检测本地Ollama模型**
- 无本地模型时无缝切换至OpenAI
- AI对话功能
- 智能分析聊天内容

### ✅ 待办事项管理
- 手动添加待办
- **AI自动从聊天内容提取待办事项**
- 优先级管理
- 完成状态跟踪

### 📚 知识库
- 将聊天记录结构化存储
- 关键词搜索
- AI智能问答

### 🎨 精美界面
- 现代化Web技术构建的GUI
- 直观友好的操作体验
- 适配非技术用户

## 🚀 快速开始

### 系统要求
- Windows 10+ / macOS 10.13+ / Linux
- Node.js 18+
- Python 3.8+ (用于读取微信数据)
- 微信PC版 4.0+（已登录）

### 安装

```bash
# 克隆仓库
git clone https://github.com/your-username/wechat-ai-assistant.git
cd wechat-ai-assistant

# 安装Node.js依赖
npm install

# 安装Python依赖（可选，用于支持压缩消息内容）
pip install -r src/python/requirements.txt

# 启动应用
npm start
```

### 开发模式

```bash
# 同时启动服务器和Electron
npm run dev
```

### 构建

```bash
# 构建Windows版本
npm run build:win

# 构建macOS版本
npm run build:mac

# 构建Linux版本
npm run build:linux
```
